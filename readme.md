## Download

### Spring Boot Starter
```xml
<dependency>
  <groupId>name.nkonev.multipart-spring-graphql</groupId>
  <artifactId>multipart-spring-graphql</artifactId>
  <version>VERSION</version>
</dependency>
```

# Compatibility

| multipart-spring-graphql | Java    | Spring Boot       |
|--------------------------|---------|-------------------|
| 0.10.x+                  | Java 8+ | Spring Boot 2.7.x |
